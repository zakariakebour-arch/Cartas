# Cartas
Mini proyecto que permite practicar JavaScript y manipulación del DOM junto con CSS. El usuario ingresa un valor numérico y, al hacer clic en el botón, se crean tantas tarjetas como indique ese número. Cada tarjeta contiene un título, una descripción y una imagen, y puede eliminarse haciendo clic sobre ella.

Características:

Creación dinámica de elementos HTML usando createElement y appendChild.

Uso de innerHTML para añadir contenido complejo a cada tarjeta.

Manejo de eventos en elementos generados dinámicamente (addEventListener).

Estilos modernos con CSS: grid, sombras, bordes redondeados y efectos hover.

Eliminación individual de tarjetas al hacer clic sobre ellas.

Cómo usarlo:

Abrir index.html en un navegador.

Ingresar un número en el input superior.

Presionar el botón "Crear tarjetas".

Hacer clic en cualquier tarjeta para eliminarla.

Tecnologías usadas:

HTML5

CSS3 (Grid, Flexbox, transiciones)

JavaScript (DOM, eventos, bucles)
